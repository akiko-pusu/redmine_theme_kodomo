# このリポジトリについて

こちらはデモ用のリポジトリです。
[Redmine](https://redmine.org/) のテーマの練習用に作成しております。

CSSについては適切でない書き方になっているため、順次修正をしていく予定です。

## 配置方法

- Redmineのディレクトリ/public/themes/ 以下に展開します
- 配置後にRedmineを再起動します

参考: Redmineのディレクトリ/public/themes/ 以下

```bash
$ tree -L 1
```
